# 宝箱

通过在战斗中击杀魔族可以获得宝箱。宝箱的获得位置参见[宝箱地图](/zh/maps)、[战斗点](/zh/battle-locations)。

每次区域危险度被重置后，仅在各个战斗点的首次战斗中可以获得宝箱。诺艾儿战败或返回据点时区域危险度将被重置，同时所有携带的宝箱被开启。

## 宝箱类型

宝箱按照颜色可以被分为以下几类：

- <font color=#B61158>水果</font>：「水果1」、「水果2」
- <font color=#CFBB69>谷物</font>：「谷物1」、「谷物+」、「甜蔬1」、「甜蔬2」
- <font color=#8B857D>素材</font>：「素材」、「素材+」、「素材++」
- <font color=#40505A>矿石</font>：「矿石1」、「矿石2」、「矿石3」、「矿石+」
- <font color=#43B82C>蔬菜</font>：「绿色蔬菜1」、「红黄色蔬菜1」、「各类蔬菜1」、「蘑菇1」、「蘑菇+」
- <font color=#369C5C>香草</font>：「香草1」、「香草2」
- <font color=#611A8A>肉类</font>：「肉类1」、「肉类+」
- <font color=#5C3DF8>宝石</font>：「煤炭1」、「火药1」、「宝石1」、「宝石+」
- <font color=#794C4C>魔族</font>：「毒水1」、「魔族素材1」、「魔族素材2」、「魔族素材3」、「魔族素材4」

## 宝箱掉落与劣化

击破战斗点可以获得的宝箱分为**普通宝箱**和**秘密宝箱**。战斗点面板上会显示该战斗点必定会掉落的普通宝箱，若面板上显示会生成污染体还会额外显示该战斗点可能由污染体掉落的秘密宝箱的图案，但是名字显示为「???」。关于各个战斗点可能掉落的宝箱的详细信息参见[战斗点列表](/zh/battle-locations#战斗点列表)。

秘密宝箱在掉落时默认有 75% 的概率劣化为同类型的普通宝箱。特殊地，[木偶复仇战](/zh/battle-locations/puppet-revenge)中掉落的秘密宝箱有 40% 的概率劣化为同类型的普通宝箱。

# 效果转轮

通过击破战斗点或使用旅馆「时光黑巢」的相应服务提高`时间危险度`以及丢弃诺艾儿体液提高`体液危险度`可以获得效果转轮。更多关于危险度的信息参见[战斗点](/zh/battle-locations#危险度)。

可以获得的效果转轮数量的计算规则如下：

<details>
  <summary>点击展开</summary>

可以获得的效果转轮数量与通过击破战斗点提高时间危险度的次数（以下简称`有效战斗数`）、`时间危险度`和`体液危险度`有关：

```c
若 (3 < 时间危险度 < 8) 则:
  时间相关数量 = (时间危险度 + 3) / 5
否则:
  时间相关数量 = (时间危险度 + 2) / 5

若 (3 < 体液危险度 / 3 < 8) 则:
  体液相关数量 = (体液危险度 / 3 + 3) / 5
否则:
  体液相关数量 = (体液危险度 / 3 + 2) / 5

可以获得的效果转轮数量 = min(255, min(时间相关数量, 有效战斗数) + min(体液相关数量, 有效战斗数 + 2))
```

其中`时间危险度`没有上限，`体液危险度`的上限为45。所有除法向下取整。

下面举例说明：

<details>
  <summary>点击展开</summary>

```c
// 有效战斗数 = 0; 时间危险度 = 0; 体液危险度 = 45
时间相关数量 = (0 + 2) / 5 = 0
体液相关数量 = (45 / 3 + 2) / 5 = 3
可以获得的效果转轮数量 = min(255, min(0, 0) + min(3, 0 + 2)) = 2

// 有效战斗数 = 3; 时间危险度 = 7; 体液危险度 = 0
时间相关数量 = (7 + 3) / 5 = 2
体液相关数量 = (0 / 3 + 2) / 5 = 0
可以获得的效果转轮数量 = min(255, min(2, 3) + min(0, 3 + 2)) = 2
```

</details>

可以获得的效果转轮种类的规律如下：

<details>
  <summary>点击展开</summary>

第 1 至 3 个：每个从以下组合中不重复随机抽取一组：
- 品质+ (100%)
- 数量+ (100%)
- 品质++ (25%) / 数量++ (25%) / 金币+ (25%) / 数量+ (25%)

第 4 个恒定为：数量X。

第 5 至 7 个：每个从以下组合中不重复随机抽取一组：
- 福袋 (100%)
- 金币+ (100%)
- 品质+ (30%) / 数量+ (70%)
- 品质++ (24%) / 数量++ (56%) / 品质+ (6%) / 数量+ (14%)

第 8 至 16 个：每三个一组，每个从以下组合中不重复随机抽取一组：
- 福袋 (50%) / 金币+ (50%)
- 品质++ (24%) / 数量++ (56%) / 品质+ (6%) / 数量+ (14%)
- 品质+++ (40%) / 数量+++ (60%)

自第 17 个起：每三个一组，每个从以下组合中不重复随机抽取一组：
- 福袋 (50%) / 金币+ (50%)
- 品质++ (24%) / 数量++ (56%) / 福袋 (6%) / 数量+ (14%)
- 品质+++ (40%) / 数量+++ (60%)
- 品质+++ (30%) / 数量+++ (70%)
- 数量X (30%) / 数量+++ (70%)

实际上，第 1 至 7 个效果转轮会在诺艾儿离开据点或战败结算时同时确定，自第 8 个起每三个一组会在将要获得该组的第一个效果转轮时同时确定，因此可以通过读取存档来改变下一组效果转轮的随机数。

</details>

效果转轮在采集素材或开启宝箱时将按照获得顺序被使用，且每次最多使用 8 个。若携带的效果转轮数大于 8 个，则采集素材时将使用最后获得的 8 个，开启宝箱时将被要求丢弃多出的数量。每次采集素材或开启宝箱时能够获得的物品数量上限为 99。

诺艾儿战败强制开启宝箱时无法使用效果转轮。特殊地，因放弃工会任务强制开启工会暂存宝箱时，工会处存有的效果转轮仍然可以使用。

诺艾儿携带的效果转轮将在区域危险度重置时被清空。

# 战斗点

在野外区域中分布着许多的战斗点。一般可以通过采摘战斗点内的魔力植物、在战斗点面板处按下确认键、或是在遭遇战的情况下闯入战斗点内来触发战斗，触发战斗后击败战斗点内出现的所有魔族即视为战斗胜利。在拓荒者山岳中也存在一些深暗的长有鲜艳绿色魔力植物老树的特殊战斗点，这些战斗点在[上升量](/zh/battle-locations#危险度)不等于[战斗点危险度](/zh/battle-locations#危险度)的时候不会显示战斗点面板，魔族和魔力植物会在其中提前生成，且击杀魔族或采集魔力植物后不会强制触发战斗。

战斗点分为**普通战斗点**和**夜间战斗点**，夜间战斗点只能在时间为夜晚的情况下触发战斗。夜间战斗点在白天的时候不会生长魔力植物，也不会显示战斗面板。特殊地，[愚者](/zh/enemy/the-fool)在且仅在编织者之森中所有的夜间战斗点出现，[皇后](/zh/enemy/empress)在且仅在拓荒者山岳中所有的夜间战斗点出现，二者均有几率掉落对应的塔罗牌。

一些战斗点的附近会有被藤蔓缠绕起来的机关门。这些机关门会在对应的战斗点被首次击破后打开。

若诺艾儿杀死了[木偶店主](/zh/enemy/sales-puppet)，则在经过一些战斗点时有几率触发[木偶复仇战](/zh/battle-locations/puppet-revenge)。

## 危险度

每个战斗点都有自己的`战斗点危险度`，在一定程度上可以体现该战斗点的基础难度。在此基础上，战斗点的战斗难度会随着`区域危险度`的提高而提高。`区域危险度`的计算规则如下：

```c
区域危险度 = min(160, 时间危险度 + 体液危险度)
```

`区域危险度`的上限为 160。诺艾儿战败、返回据点或使用旅馆「时光黑巢」的标准套房服务时`时间危险度`和`体液危险度`将被重置，`区域危险度`也随之被重置。

### 时间危险度

`时间危险度`决定了游戏中的时间。每 16 `时间危险度`为 1 天，即 0 至 15 `时间危险度`为第 1 天，16 至 31 `时间危险度`为第 2 天，依此类推。若`时间危险度 mod 16 < 9`则时间为白天，否则时间为夜晚，`mod`表示取模运算。换言之，每天的前 9 `时间危险度`为昼，后 7 `时间危险度`为夜。

通过击破战斗点可以提高`时间危险度`，能够提高的量被称为`上升量`。这个数值会显示在战斗点面板上，除非`上升量`就等于`战斗点危险度`。`上升量`的计算规则如下：

```c
若 (区域危险度重置后该战斗点还未被击破) 则:
  上升量 = 战斗点危险度
否则:
  若 ((当前天数 >= 4) 或 (出门时的天数 >= 2)) 则:
    上升量 = max(0, min(战斗点危险度, 上次击破该战斗点后进行的有效战斗次数 - 1))
  否则:
    上升量 = max(0, min(战斗点危险度, 上次击破该战斗点后进行的有效战斗次数))
```

其中`有效战斗`指的是`上升量`不为 0 的战斗，即会提高`时间危险度`并推进时间的战斗。简单来说，通过击破其他战斗点推进时间可以提高已击破的战斗点的`上升量`。`上升量`的下限为 0，上限为`战斗点危险度`。

除战斗外，使用旅馆「时光黑巢」的帐篷地铺或经济标间服务也可以提高`时间危险度`。

### 体液危险度

通过丢弃诺艾儿体液可以提高`体液危险度`，能够提高的量为丢弃的诺艾儿体液的品质等级之和。`体液危险度`的上限为 45。

### 工会任务危险度补正

一些工会任务会附带`危险度补正`，这个数值会在执行该任务时临时计入`区域危险度`。

## 遭遇战

进入战斗点时可能会触发**遭遇战**。遭遇战的特征是看不到战斗点面板，且一些在战斗开始后才会生长的魔力植物会提前生长，此时闯入战斗点内的话将有很大概率会直接开始战斗。遭遇战的触发遵循以下规律：

- 如果一个夜间战斗点从未被击破过，那么在初次遇到的时候必定触发遭遇战。
- 教程关卡不会触发遭遇战。
- 战斗点「森之领主」不会触发遭遇战。
- 第 1 至 3 天的白天不会触发遭遇战。
- 已被击破的战斗点在第 1 至 2 天内不会触发遭遇战。
- 战斗点的`上升量`小于`战斗点危险度`时不会触发遭遇战。

在满足遭遇战触发条件的基础上，遭遇战触发概率的计算规则为：

```c
天数系数 = max(0, min((当前天数 - 1) / 4, 1))
若 (当前天气为雾天或浓雾) 则:
  触发概率 = 天数系数 * 0.35 + 0.4
否则:
  触发概率 = 天数系数 * 0.175 + 0.2
```

简单来说，遭遇战的触发概率会随着天数推进而提升，从第 1 天的 20% 开始，直到进入第 5 天（64 `时间危险度`）后达到最大值 37.5%。雾天或浓雾天气会使遭遇战的触发概率翻倍。

## 宝箱

每次区域危险度被重置后，仅在各个战斗点的首次战斗中可以获得宝箱。宝箱分为**普通宝箱**和**秘密宝箱**。战斗点面板上会显示该战斗点必定会掉落的普通宝箱，若面板上显示会生成污染体还会额外显示该战斗点可能会掉落的秘密宝箱的图案，但是名字显示为「???」。

秘密宝箱在掉落时默认有 75% 的概率劣化为同类型的普通宝箱。特殊地，[木偶复仇战](/zh/battle-locations/puppet-revenge)中掉落的秘密宝箱有 40% 的概率劣化为同类型的普通宝箱。关于宝箱类型的详细信息参见[宝箱转轮](/zh/chest-reels)。
